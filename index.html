Features

Übersichtskarte

Die Übersichtskarte der Stadt Wien zeigt die Linien von U-Bahn, Tram, VRT und Bus an.

Über einen Klick auf eine Haltestelle kommt man ganz einfach zu deren Abfahrtsmonitor.

Live-Abfahrtsmonitor

Der Live-Abfahrtsmonitor zeigt die nächsten Abfahrten für eine Haltestelle an.

In regelmäßigen Abständen werden Echtzeit-Daten von den Wiener Linien abgerufen.

Optional verhalten sich die Displays wie ein echtes, altes Bahnhofsdisplay!

Ausblenden einzelner Verkehrsmittel

Sowohl auf der Karte als auch in einem Abfahrtsmonitor können Verkehrsmittel ausgeblendet werden.

So werden nur noch die gewünschten Transportmittel angezeigt!

Verschiedene Stile

Um den Vienna Traffic Monitor an Ihre individuellen Bedürfnisse anzupassen, können sie zwischen einem hellen und einem dunklen Stil wählen.
